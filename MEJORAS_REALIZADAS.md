# Mejoras implementadas en este zip

## 1. Mejoras funcionales
- Ahora los cursos creados sí quedan guardados localmente.
- Los cursos publicados aparecen en resultados de búsqueda, detalle y perfil.
- Se puede guardar un curso desde la tarjeta o desde el detalle.
- Se puede iniciar un curso y queda reflejado en el perfil.

## 2. Búsqueda y filtros
- La búsqueda conserva mejor el estado.
- Los filtros ahora viven en la URL.
- Se agregó ordenamiento por relevancia, valoración, reciente, duración y precio.
- Se agregaron chips para quitar filtros activos más rápido.

## 3. Formulario de crear curso
- Validación por campos.
- Validación de imagen.
- Validación de precio.
- Validación de lecciones.
- Campo de etiquetas.
- Sugerencia automática de duración según las lecciones.
- Publicación con feedback visual mejorado.

## 4. Perfil
- Sección de cursos publicados.
- Sección de guardados conectada al guardado real.
- Cursos iniciados visibles en progreso.

## 5. Calidad general
- Mejoras pequeñas de accesibilidad.
- Imágenes con `loading="lazy"` en tarjetas.
- Ajustes de estructura para que el proyecto quede más escalable.
- Se agregó un módulo de utilidades para centralizar lógica del catálogo.

## Lenguajes y herramientas principales
- TypeScript
- React
- Tailwind CSS
- Vite
